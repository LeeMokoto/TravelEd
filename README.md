# TravelEd

A personal travel app, built on top of the open-source **Wonderous** showcase app
(by [gskinner](https://gskinner.com/) in partnership with the Flutter team, MIT licensed).

TravelEd starts from Wonderous as a base and expands it into a functional, personal
travel companion. See `README_WONDEROUS.md` for the original project's documentation
and attribution.

## Status
Early stage. Using Wonderous as the foundation and iterating from there.

## Getting started
```
flutter pub get
flutter run
```
